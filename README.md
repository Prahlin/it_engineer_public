# LMS Troubleshooting Scenario #1

## Issue Summary  
An instructor reported that a student’s grade is not visible in the gradebook.
However, the student claims they submitted the assignment.

## Steps Taken  
- Checked the student’s submission status via the API.  
- Queried the database to see if the grade was recorded.  
- Verified if the instructor had the correct permissions.

## Outcome  
The submission was found but was not graded.
Instructor was guided to review pending submissions.
