# LMS TROUBLESHOOTING: STUDENT UNABLE TO LOG INTO LMS

## (See `login_sample_submission_response.json` for example LMS API responses used during troubleshooting)

## Issue Description

## A student reported that they are unable to log into the LMS.
## The instructor confirmed that the student should have access to the course but the student receives a login failure message.




## Troubleshooting Steps:

## 1. Verify that the LMS login service is reachable through the LMS APIs

## 2. Confirm that the student account exists in the LMS user system

## 3. Verify that the student’s LMS account status is active

## 4. Test the LMS authentication process using the student’s login credentials

## 5. Check whether the LMS APIs are limiting login attempts due to rate limits or throttling




## Resolution

## The issue was resolved by reactivating the student’s LMS account, allowing the student to successfully authenticate and log into the LMS.
